vmchecker

About

vmchecker is a long running project within the University POLITEHNICA of Bucharest. It is a system used for automatically testing and grading student assignments. The backend is written in Python and the current interface is written using the Java GWT.

Update new UI

We need to get the new UI up to date and implement all of the missing features in order to replace the old UI. We would also like to have vmchecker use a continuous integration tool such as travis-ci. This will allow development to go on much faster.

First steps: Start User Documentation

  • Basic knowledge of Python is preferable
  • Experience with Node.js and Meteor.js is a plus
